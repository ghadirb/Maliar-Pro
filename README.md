# Maliar Pro - دستیار مالی هوشمند

یک برنامه حرفه‌ای مدیریت مالی و یادآوری با هوش مصنوعی برای اندروید

## ویژگی‌ها

### 📊 حسابداری پیشرفته
- مدیریت درآمدها و هزینه‌ها
- مدیریت چک‌ها و اقساط
- محاسبه تراز کل
- گزارش‌های مالی

### 📋 یادآوری هوشمند
- یادآوری‌های ساده و تکرار شونده
- اتصال خودکار به چک‌ها و اقساط
- دسته‌بندی و اولویت‌بندی
- اعلان‌های هوشمند

### 🤖 دستیار مالی هوشمند
- تحلیل مالی شخصی‌سازی شده
- پیشنهادهای صرفه‌جویی
- پیشنهادهای سرمایه‌گذاری
- گفتگوی طبیعی با هوش مصنوعی
- سوالات پیشنهادی هوشمند

### 💰 وضعیت مالی جامع
- مدیریت دارایی‌ها (نقد، بانک، طلا، ارز، سهام، ملک و...)
- مدیریت بدهی‌ها (وام، چک، کارت اعتباری و...)
- تعیین اهداف مالی
- ثبت درآمدهای ثابت
- ترجیحات مالی شخصی
- تنظیمات دسترسی هوش مصنوعی

### 👤 پروفایل
- مدیریت کلیدهای API با رمزگذاری خودکار
- تنظیمات برنامه
- مدیریت مخاطبین با شماره ردیف
- تماس صوتی با تشخیص صدا

## معماری

### ساختار پایگاه داده
- **Contact**: مخاطبین با شماره ردیف
- **Income/Expense**: درآمدها و هزینه‌ها
- **Check**: مدیریت چک‌ها
- **Installment**: مدیریت اقساط
- **Reminder**: یادآوری‌ها با اتصال به حسابداری
- **Asset**: دارایی‌های متنوع
- **Debt**: بدهی‌ها
- **FinancialGoal**: اهداف مالی
- **FixedIncome**: درآمدهای ثابت
- **FinancialPreferences**: ترجیحات مالی

### مدیریت کلیدهای API
- بارگذاری خودکار از منابع رمزگذاری شده
- پشتیبانی از چندین ارائه‌دهنده (Liara, GAPGPT, OpenAI, و...)
- رمزگذاری AES-GCM
- مکانیزم فال‌بک برای اطمینان از دسترسی

## پیش‌نیازها

- Android Studio Hedgehog یا جدیدتر
- JDK 17
- Android SDK 34
- Gradle 8.2

## نصب و راه‌اندازی

1. کلون کردن پروژه:
```bash
git clone https://github.com/ghadirb/Maliar-Pro.git
cd Maliar-Pro
```

2. باز کردن پروژه در Android Studio

3. همگام‌سازی Gradle:
```bash
./gradlew sync
```

4. بیلد پروژه:
```bash
./gradlew build
```

5. اجرا روی دستگاه یا شبیه‌ساز

## پیکربندی

### کلیدهای API
کلیدهای API به صورت خودکار از منابع رمزگذاری شده بارگذاری می‌شوند. برای استفاده از کلیدهای اختصاصی:

1. به تب پروفایل بروید
2. بخش کلیدهای API را انتخاب کنید
3. کلیدهای خود را اضافه کنید

### دسترسی هوش مصنوعی
در بخش وضعیت مالی می‌توانید تنظیم کنید:
- آیا دستیار به حسابداری دسترسی داشته باشد؟
- آیا از دارایی‌ها در تحلیل استفاده شود؟
- آیا اهداف مالی در پیشنهادها لحاظ شوند؟
- نوع پیشنهادها (سرمایه‌گذاری، صرفه‌جویی، خرید، برنامه غذایی)

## ساختار پروژه

```
app/
├── src/main/
│   ├── java/com/maliar/pro/
│   │   ├── database/          # پایگاه داده Room
│   │   ├── models/            # مدل‌های داده
│   │   ├── ui/                # رابط کاربری
│   │   │   ├── accounting/    # حسابداری
│   │   │   ├── reminders/     # یادآوری
│   │   │   ├── assistant/     # دستیار هوشمند
│   │   │   ├── financial/     # وضعیت مالی
│   │   │   └── profile/       # پروفایل
│   │   ├── utils/             # ابزارها
│   │   └── MaliarProApplication.kt
│   ├── res/                   # منابع
│   └── AndroidManifest.xml
├── build.gradle.kts
└── proguard-rules.pro
```

## تکنولوژی‌های استفاده شده

- **Kotlin** - زبان اصلی
- **Jetpack Compose** - رابط کاربری مدرن
- **Room Database** - پایگاه داده محلی
- **Coroutines & Flow** - برنامه‌نویسی ناهمگام
- **Material Design 3** - طراحی رابط کاربری
- **Navigation Component** - ناوبری
- **OkHttp** - درخواست‌های شبکه
- **Gson** - پردازش JSON
- **Work Manager** - کارهای پس‌زمینه
